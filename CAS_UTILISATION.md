## ⚠️ CAS 10: RÉSOUDRE LES PROBLÈMES

### ❌ App Ne Démarre Pas?
```bash
pip install -r requirements.txt
python lancer_app.py
```

### ❌ Port Déjà Utilisé?
```bash
streamlit run app.py --server.port 8502
```

### ❌ Pas de Résultats?
- Écrivez une requête plus détaillée
- Ajoutez des mots-clés spécifiques
- Évitez requêtes trop vagues

### ❌ App Très Lente?
- Réduisez le nombre de résultats
- Vérifiez votre connexion
- Fermez autres applications
